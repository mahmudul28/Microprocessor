Segment Overlap Rule

The 8086 has a 20-bit address bus (1MB memory) but 16-bit registers. To reach 1MB, it uses 64KB segments shifted by 16 bytes (10H).

Important: Because segments shift by only 10H, they heavily overlap. E.g., 1000:2000 and 1200:0000 point to the exact same physical memory address (12000H).

Little Endian Format

When saving a 16-bit word (e.g., AABB) into memory, the "Little End" (Least Significant Byte) goes to the lower address.
